# skill-scaffold

## 这个 Skill 是什么？

这是一个指导你为规范文档自动生成标准 Agent Skills 的 Skill。它会读取规范文档，提取主题与触发场景，产出符合统一格式的 Skill 文件夹结构，包括 SKILL.md 和 resources/ 文件夹。

## 什么时候使用？

当你有一个 `<规范文档>.md`时，需要为其创建 `SKILL.md` 和 `resources/` 文件夹并引用其中的规范时。
