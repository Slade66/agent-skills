# Schema ASCII Table

请为 .sql 中每个 `CREATE TABLE` 上面添加一张 ASCII 注释表，用于直观地解释这个表的每个字段以及存储的数据长什么样。

要求：
2. 所有 ASCII 行都用 SQL 注释 `-- ` 开头。
3. 表头格式：“字段名（中文含义） 概括类型”。
4. 表头必须横向包含该表全部字段。
5. 表头下方写 3 行 mock 数据。

示例：
-- +------------------+--------------------------+----------------------------+
-- | id（用户ID） 整数 | name（姓名） 字符串       | created_at（创建时间） 时间  |
-- +------------------+--------------------------+----------------------------+
-- | 1                | 张三                     | 默认当前时间                |
-- | 2                | 李四                     | 默认当前时间                |
-- | 3                | 王五                     | 默认当前时间                |
-- +------------------+--------------------------+----------------------------+
