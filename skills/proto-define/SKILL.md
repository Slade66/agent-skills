# proto-define

## 这个 Skill 是什么？

这是一个指导你在项目中编写符合规范的 *Protobuf API 定义 的 Skill。它覆盖了 proto 文件的目录组织、命名规则、HTTP 映射约定、资源标识、请求响应设计、时间字段、错误码枚举、版本演进策略等全部规范要求。

当你需要为项目新增一个 gRPC/HTTP 接口的资源定义时，此 Skill 提供可复制的 proto 模板和约束检查清单。

## 什么时候使用？

- 需要为项目新增资源的 `.proto` 接口定义
